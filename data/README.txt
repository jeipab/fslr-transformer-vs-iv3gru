This directory is used to store project data.

Subfolders:
- raw: original, unmodified data
- processed: cleaned/processed data ready for use

File types:
- raw: original files (e.g., .csv, .txt, .wav, .jpg, .json, .zip)
- processed: derived/ready-to-use files (e.g., .npz, .npy, .pt, .pkl, cleaned .csv)
